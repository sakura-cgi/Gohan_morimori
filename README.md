<h2>🎮 Overview</h2>

<p>
「Thermo Climber」は、体温を管理しながら雪山を攻略する
2Dアクションゲームです。

プレイヤーは宇宙人となり、服の脱ぎ着によって体温を調節します。

寒すぎると氷属性、
暑すぎると炎属性の攻撃が可能になりますが、
どちらも体力や移動能力にデメリットがあります。

敵の属性や地形に応じて体温を調整し、
最適な状態を維持しながら山頂を目指します。
</p>
